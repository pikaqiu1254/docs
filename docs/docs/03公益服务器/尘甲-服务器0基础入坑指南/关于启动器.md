## **PCL2**

全称：“Plain Craft Launcher 2”  

:::tip 提示
PCL2 启动器的开发者为“龙腾猫跃”**个人**  
在此特别鸣谢  
[龙腾猫跃（bilibili）](https://space.bilibili.com/11343203) [龙腾猫跃（爱发电）](https://afdian.com/a/LTCat)[PCL2在github上的仓库](https://github.com/Meloong-Git/PCL)
:::

PCL2启动器适用于win10+的windows系统，win7系统需注意系统版本为六十四位版本。(win7以下系统不支持64位java，不推荐游玩minecraft较高版本)

## **HMCL**

HMCL启动器，全称“Hello Minecraft! Launcher”  
官网地址：[点我进入HMCL官网](https://hmcl.huangyuhui.net/)  
此启动器支持windows，macOS以及Linux特定版本  

## **BakaXL**

官网：[点我进入BakaXL官网](https://www.bakaxl.com/)  
仅支持windows,同时国内首个支持 Quilt 和 NeoForge 模组加载器的启动器!!,但一般不推荐使用!!。

## **XMCL**

XMCL启动器：全程“X Minecraft Launcher”  
官网地址：[点我进入XMCL官网](https://xmcl.app/zh/)
此启动器支持windows，macOS以及Linux特定版本
