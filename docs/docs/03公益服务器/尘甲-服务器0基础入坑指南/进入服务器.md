在**主界面**点击**多人游戏**进入二级菜单后点击**添加服务器**
![示例](/img/03公益服务器/尘甲-服务器0基础入坑指南/进入服务器/01.png)
填写你需要游玩的服务器，的 IP 地址（第二栏），第一栏可以任意填写  
点击**完成**
![示例](/img/03公益服务器/尘甲-服务器0基础入坑指南/进入服务器/02.png)
此处以四周目服务器为例，双击你想进入的服务器图标，即可进入游戏。
