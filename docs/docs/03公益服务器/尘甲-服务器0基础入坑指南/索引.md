## **账号**
如果你要登入到服务器，首先你需要拥有一个服务器接受的MC账户  
服务器目前仅支持**正版账号**和**第三方登陆（littleSkin 皮肤站账号）**  
如果需要帮助，请查看[关于账号](关于账号.md)

## **启动器**
